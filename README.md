# 재난현장 탐색 RC-Car

> 재난현장 정보수집용 RC-Car

## 프로젝트 개요

자연재해나 인적 재난 상황에는 가스 누출, 건물 붕괴, 폭발 등 방재 요원들의 안전을 위협하는 요소들이 곳곳에 산재해 있다.
따라서 방재 요원의 안전을 확보하고 재난 현장의 정보 수집을 위해 재난 현장 정보 수집을 위한 장비가 필요하다.
